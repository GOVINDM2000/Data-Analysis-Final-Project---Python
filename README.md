Project Title: District-Wise IPC Crimes 2022 - EDA.
Description: An Exploratory Data Analysis (EDA) of 2022 India district-wise IPC crimes using Python.
Technologies Used:Python for data analysis.Pandas & NumPy for data processing,Matplotlib & Seaborn for data visualizations.
Visualizations & Core Findings:
Bar Chart (sns.barplot): Directly isolates and ranks the top 10 states with the highest overall crime rates.
Box Plots (sns.boxplot): Showcases district-level variation and highlights extreme high-density crime outliers.
Heatmap (sns.heatmap): Maps out correlation patterns between domestic and public offense categories.
